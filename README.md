# FlightPHP + Alpine AJAX Template

A minimal template demonstrating [FlightPHP](https://flightphp.com/) with [Alpine AJAX](https://alpine-ajax.js.org/) and [Latte](https://latte.nette.org/) templates.

## Setup

```bash
composer install
npm install
npm start
```

Open [http://localhost:8000](http://localhost:8000)
